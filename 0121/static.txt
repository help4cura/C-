static 키워드

선언 시, static 키워드를 반드시 사용해야 한다.
한 번 할당되면 메모리와 값이 프로그램이 종료될 때 까지 유지된다.
초기화는 단 한 번만 할 수 있으며, 초기화는 반드시 소스 파일에서 진행되어야 한다.
초기화를 하지 않은 상태로 어느 곳에서도 사용되지 않으면, 메모리가 할당되지 않는다.

- 전역 변수에서 선언한 static 키워드
  ■ 선언된 파일 내에서만 참조를 허용한다.

- 함수 내에 선언한 static 키워드
  ■ 한번만 초기화되고, 지역 변수와 다르게 함수를 탈출해도 소멸되지 않는다.

클래스에 static 키워드를 선언하면, 클래스 외부에도 반드시 선언해 주어야 한다.
